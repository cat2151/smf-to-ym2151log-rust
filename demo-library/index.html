セットアップ手順 (Rust不要)

GitHub Pages に配置されたビルド済み WASM（pkg/smf_to_ym2151log.js, .wasm, .d.ts）を直接ダウンロードします。ローカルで Rust や wasm-pack を入れる必要はありません。

  1. npm install github:cat2151/smf-to-ym2151log-rust
  2. npm run dev で開発サーバーを起動（ポート8001）
  3. 本番ビルドは npm run build
